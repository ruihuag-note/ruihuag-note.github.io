# outline-offset

- 对轮廓进行偏移，并在超出边框边缘的位置绘制轮廓

- 规定边框边缘之外 15 像素处的轮廓：

```css
div {
  border: 2px solid black;
  outline: 2px solid red;
  outline-offset: 15px;
}
```
