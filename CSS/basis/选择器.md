# 选择器

| 属性       | 选择器                  |
| ---------- | ----------------------- |
| html元素   | 直接html标签, h1 span p |
| id         | #                       |
| class      | .                       |
| type       | [type="属性"]           |
| !important | 最高级选择器            |

## `[attr](属性)`

用法:

1. `[attr](有该属性)`
2. [attr =xxx] (属性值 是xxx)
3. `[attr^=xxx](属性是xxx开头)`
4. `[attr$=xxx](属性是xxx结尾)`
5. `[attr*=xxx](属性包含xxx)`

## 权重计算

| 选择器                    | 权重  |
| ------------------------- | ----- |
| !important                | >1000 |
| 内联(style='color: #fff') | 1000  |
| id                        | 100   |
| class                     | 10    |
| 伪类                      | 10    |
| 元素                      | 1     |
| *                         | 0     |
| 继承                      | 无    |

## 选择器的使用

> 内部样式和外部样式   采用就近原则

| 选择器         | 书写                                                         |
| -------------- | ------------------------------------------------------------ |
| 后代选择器     | `h1 span`                                                    |
| 子代选择器     | `h1>span`                                                    |
| 伪类选择器     | `:hover  :link :visted  :active`                             |
| 相邻兄弟选择器 | `span+span`                                                  |
| 通用兄弟选择器 | `p~span`( 不一定要紧跟着, 但是必须要有一个共同的父元素 )<br>`p`元素之后的所有`span`元素 |
| 通用选择器     | `*`                                                            |
| 群组选择器     | `span, p, h1`                                                  |
| 结构伪类选择器 | `:first-child  : nth-child(n)    : last-child`             |
| 伪元素选择器   | `:after    :before`                                            |
